# 03-01 Plan

## Goal

Connect the one-battle prototype into the full 10-battle MVP run.

## Scope

- Add a `RunController` that advances through battle, reward, victory, and game-over stages.
- Add a pure `RewardService` for deterministic reward choice generation and reward application.
- Build reward and result screens and route home-screen starts into the run flow.
- Add tests for reward application, run progression, and the reward transition UI.

## Verification

- `cd app && flutter analyze`
- `cd app && flutter test`
- Confirm wins route to rewards, rewards apply immediately, losses route to game over, and battle 10 can end in victory.
